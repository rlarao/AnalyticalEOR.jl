AnalyticalEOR.jl: Instantaneous Spectral Analysis in Julia

Installation

Users

  1. Download Julia v1.6.4 or later, if you haven't already.
  2. Add AnalyticalEOR package by entering the following at the REPL ]add https://github.com/rlarao/AnalyticalEOR.jl.

Developers

  1. Clone the AnalyticalEOR.jl repository to username/.julia/dev/.
  2. Enter the package manager in REPL by pressing ], then add the package by typing dev AnalyticalEOR.
